📦 Order Tracking System
📌 Problem Statement

Develop a Java-based Order Tracking System that records customer orders, updates order status (Pending, Shipped, Delivered), and displays order details and current status using appropriate classes, constructors, and methods.

🚀 Project Overview

The Order Tracking System is a simple web application built using HTML, CSS, and JavaScript.
It allows users to manage customer orders efficiently by performing CRUD operations (Create, Read, Update, Delete).
The system uses browser localStorage to store data permanently without a backend database.

✨ Features
➕ Add new customer orders
🔄 Update order status (Pending / Shipped / Delivered)
🔍 Search order by Order ID
🗑 Delete orders
💾 Persistent storage using localStorage
🎨 Simple and responsive UI
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
localStorage (Browser Storage)

▶️ How to Run the Project
Download or clone the repository
Open the folder in VS Code
Locate index.html
Double-click to open in browser
Start using the application

⚙️ How It Works
Orders are stored as JavaScript objects
Each order contains:
Order ID
Customer Name
Product Name
Status
Data is saved using localStorage
Functions handle:
Add Order
Update Status
Search Order
Delete Order
